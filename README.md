# 🇲🇲 MBTI Personality Test (Burmese Version)

ဤ Project သည် လူသားတို့၏ ကိုယ်ရည်ကိုယ်သွေးနှင့် စိတ်နေစိတ်ထားများကို ခွဲခြားသတ်မှတ်ပေးသော **MBTI (Myers-Briggs Type Indicator)** စစ်ဆေးမှုကို မြန်မာဘာသာဖြင့် အလွယ်တကူ ပြုလုပ်နိုင်ရန် ဖန်တီးထားသည့် Web Application တစ်ခု ဖြစ်ပါသည်။

---

## ✨ အဓိက ပါဝင်သော Feature များ

- **User-Friendly UI:** Web ရော Mobile မှာပါ သေချာမြင်ရစေရန် High Contrast Design ဖြင့် ပြုလုပ်ထားပါသည်။
- **Responsive Design:** မည်သည့် Device မျိုးနှင့်မဆို အချိုးအစားကျကျ အသုံးပြုနိုင်ပါသည်။
- **20 Precise Questions:** MBTI Dimensions ၄ ခုလုံးကို တိကျစွာ ခွဲခြမ်းစိတ်ဖြာပေးနိုင်မည့် မေးခွန်း ၂၀ ပါဝင်ပါသည်။
- **Instant Results:** စစ်ဆေးမှုအပြီးတွင် မိမိ၏ Type နှင့် ၎င်း၏ အသေးစိတ် ဖော်ပြချက်များကို ချက်ချင်း ကြည့်ရှုနိုင်ပါသည်။

---

## 🚀 စတင်အသုံးပြုပုံ (How to Run)

ဤ Project သည် HTML/CSS/JS သီးသန့် (Static Site) ဖြစ်သောကြောင့် အသုံးပြုရန် အလွန်လွယ်ကူပါသည် -

1. Repository ကို **Download/Clone** လုပ်ပါ။
2. `index.html` ဖိုင်ကို Browser ဖြင့် ဖွင့်ပါ။
3. "စစ်ဆေးမှု စတင်ရန်" ကို နှိပ်၍ မေးခွန်းများကို ဖြေဆိုပါ။

---

## 🛠 နည်းပညာပိုင်းဆိုင်ရာ အချက်အလက်များ

- **Frontend:** HTML5, CSS3 (Flexbox, Media Queries)
- **Logic:** Vanilla JavaScript
- **Fonts:** Google Fonts (Padauk) - မြန်မာဘာသာ ဖတ်ရှုရ အဆင်ပြေစေရန်

---

## ⚖️ လိုက်နာရန် အချက်များ

ဤစစ်ဆေးမှုသည် ပညာပေးရန်နှင့် မိမိကိုယ်မိမိ ပိုမိုနားလည်စေရန်သာ ရည်ရွယ်ပါသည်။ ပညာရပ်ဆိုင်ရာ စိတ်ပညာဆိုင်ရာ အကြံပေးချက်များအဖြစ် အပြည့်အဝ အသုံးမပြုသင့်ပါ။
